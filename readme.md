# 1. きいてきいて！　クイズ$$ファミリオネア

## 2. 課題内容（どんなアプリか）
 - 親子の日々の対話を楽しく豊かにするアプリ 
 - 家族でクイズのお題を考え、AIの力を借りてクイズを生成、クイズミリオネアのようなUIでクイズを楽しめます。
 - ホストはお題を自由に選べます。家族問題を忍び込ませることもできます。
 - 回答者の年齢設定にあわせて出題レベルが調整されます。
 - 親視点・・・子どもの興味関心を引き出したり広げたいテーマをクイズという形で一緒に楽しく学べます。
 - 子視点・・・自分の好きなトピックをクイズにして、親に解いてもらうというわくわく感、幸せな時間が流れます。
 - クイズの軌跡はすべてアーカイブ化。将来、量的にも質的にも振り返ることができます。
 - どんな問題が出てくるかわからない（時々ＡＩ間違える）のでかなりドキドキ感ある感じになりました

 ## 5. 工夫した点、こだわった点
 - ユーザーヒアリングをしていくなかで、「子供の話をもっと広げてあげたい」とか「さりげなく親が今伝えたいことを伝えたい」といったささやかな想いを、楽しい演出で実現できないかな、と考えました。
 - 親子が出題者にも回答者になれる設計
 - 一つの画面を見て、会話が弾み、ドキドキするようなシーンを生み出したいと思いながら作りました
 - 「ファイナルアンサー？」と言い合えるように、2回クリックで正解が出ます。また、後半にいくごとに「タメ」が長くなります。
 - 家庭のオリジナルルールで遊べるように、クイズ部分はシンプルな構成にしました。
